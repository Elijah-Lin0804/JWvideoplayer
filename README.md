# JW 雙語學習播放器 (JW Bilingual Learning Player)

一個輕量、乾淨且無廣告的網頁影片播放器，專為語言學習者設計。使用者只需貼上 JW 官方影片的分享網址，程式便會全自動解析 API，實現**「英文原音發音 + 中文字幕同步」**的整合播放介面。

## 🌟 核心特色
- **一鍵自動解析**：免去繁瑣的抓包流程，貼上包含 `docid` 或 `lank` 的分享網址即可自動載入。
- **雙語無縫整合**：自動前往官方 API 提取英文發音影片與中文字幕檔案。
- **強固的防禦機制**：
  - 完整支援官方的**延遲載入机制 (`__deferred`)**，自動進行二次請求撈取核心資料。
  - 內建**正則表達式（Regex）數字偵測法**，不論分享網址如何變形，皆能精準剝離出合法的 `docid`。
- **原生高效體驗**：採用 HTML5 原生 `<video>` 標籤載入，效能優異，並支援自定義字幕樣式。

## 🛠️ 技術棧 (Tech Stack)
- HTML5 (原生影音控制、`crossorigin` 跨網域字幕掛載)
- CSS3 (Flexbox 響應式版面、自定義 `::cue` 字幕樣式、深色劇院主題)
- JavaScript (ES6+, Async/Await 異步網路請求、URLSearchParams、正則表達式)

## 🚀 如何使用
1. 將本專案下載或複製（Clone）到本機電腦。
2. 點擊 `index.html` 在瀏覽器中打開。
3. 貼上 JW 網站的影片分享連結（例如：`https://www.jw.org/finder?wtlocale=CH&docid=502200156&srcid=share`）。
4. 點擊「載入影片」按鈕，即可開始享受高畫質的英文發音與中文字幕學習環境！
